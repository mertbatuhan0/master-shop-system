MasterShopSystem 🛒
A Python-based retail management simulation that demonstrates the core principles of Object-Oriented Programming (OOP). This project simulates a shop environment where products are categorized, customers are managed, and sales transactions are processed.

🛠️ Technical Implementation (What I Practiced)
In this project, I focused on applying advanced programming concepts:

Inheritance: Created a base Product class and extended it into specialized classes like Electronic, showing how to reuse code efficiently.

Encapsulation: Managed data within classes to ensure that shop and customer information is handled securely.

Association: Built relationships between Shop, Product, and Customer objects to simulate a real-world shopping flow.

List Management: Implemented dynamic list handling to manage products and customer carts.

✨ Features
Dynamic Product Catalog: Support for different product types with specific attributes.

Shopping Cart Logic: Customers can add products to their cart and proceed to checkout.

Shop Management: Centralized system to track available products and handle customer interactions.
